## Screenshot Guidelines

- Resolution: 1920x1080 or higher
- Format: PNG (for better quality)
- Show realistic data/detections
- Include UI elements clearly
- Capture different screen states:
  - Normal state
  - With data/content
  - Interactive elements

## How to Take Screenshots

### Windows
- Press `Win + Shift + S` for snipping tool
- Or use `Win + Print Screen` for full screen

### Recommended Tool
- Use Snagit, Greenshot, or ShareX for professional screenshots
- Add annotations/highlights if needed
